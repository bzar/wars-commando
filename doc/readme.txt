Wars: Commando
------------------

Credits
-----------

Created by Teemu Erkkola (B-ZaR) for the "Pandora Homebrew Platforming Competition" 2011.
Music by Ashley Mayhem, "NES Tunes / SID Tunes" album


Controls
------------

Pandora:

    D-pad:  Move / aim grenade
    X:      Jump/OK
    A:      Shoot
    B:      Grenade
    Y:      Bazooka
    Q:      Pause

Other:

    Cursor keys:    Move / aim grenade
    S:              Jump / OK
    D:              Shoot
    A:              Grenade
    W:              Bazooka
    Esc:            Pause


Tap left/right twice to run
