_Building wars-commando on Mac OS X_
author: badger
email: janne.liuhtonen@iki.fi

Prerequisites:
-Qt >= 4.7.2 installed as a framework. For 10.4 compatibility you need to use Qt for Carbon.
-SDL and SDL_mixer -frameworks

Build:
1. qmake -spec macx-g++
2. create SDL-directory and copy SDL headers there
   mkdir SDL
   cp /Library/Frameworks/SDL.framework/Headers/*.h SDL
   cp /Library/Frameworks/SDL_mixer.framework/Headers/*.h SDL
3. make
4. macdeployqt wars-commando.app (you can ignore errors when using Carbon Qt)
5. Copy SDL-libs to the app bundle
   cp -R /Library/Frameworks/SDL.framework wars-commando.app/Contents/Frameworks
   cp -R /Library/Frameworks/SDL_mixer.framework wars-commando.app/Contents/Frameworks
6. Copy resources to the app bundle
   cp -R music wars-commando.app/Contents/Resources
   cp -R sfx wars-commando.app/Contents/Resources
7. Check that the compiled game uses frameworks that are bundled with the app. The libs are listed twice because the executable has been built for i386 and ppc architectures.
   otool -L wars-commando.app/Contents/MacOS/wars-commando
8. copy img/icon.icns to wars-commando.app/Contents/Resources
9. Edit wars-commando.app/Contents/Info.plist, add CFBundleIconFile and LSMinimumSystemVersionByArchitecture -elements
--
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist SYSTEM "file://localhost/System/Library/DTDs/PropertyList.dtd">
<plist version="0.9">
<dict>
        <key>CFBundleIconFile</key>
        <string>icon.icns</string>
        <key>CFBundlePackageType</key>
        <string>APPL</string>
    <key>CFBundleGetInfoString</key>
        <string>Created by Qt/QMake</string>
        <key>CFBundleSignature</key>
        <string>????</string>
        <key>CFBundleExecutable</key>
        <string>wars-commando</string>
        <key>CFBundleIdentifier</key>
        <string>com.yourcompany.wars-commando</string>
        <key>NOTE</key>
        <string>This file was generated by Qt/QMake.</string>
        <key>LSMinimumSystemVersionByArchitecture</key>
        <dict>
                <key>x86_64</key>
                <string>10.6</string>
                <key>i386</key>
                <string>10.4</string>
                <key>ppc</key>
                <string>10.4</string>
        </dict>
</dict>
</plist>
--

10. Done!
